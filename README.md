# IPL Match Winner Predictor

This project predicts the winner of IPL cricket matches using Machine Learning.

## Features
- Team statistics
- Head-to-head records
- Venue performance
- Toss information

## Machine Learning Model
Gradient Boosting Classifier

## Accuracy
~57%

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run

Install dependencies

pip install pandas numpy scikit-learn streamlit

Run the app

streamlit run app.py
